🦾 진행한 부분
기능 Process
기능 Process
- 영상 업로드 → 얼굴 인식 + crop 하여 저장 → crop된 이미지에 원하는 얼굴 변형 적용하여 저장 → 변형된 crop 이미지를 다시 전체 이미지에 삽입하여 저장
- 한 프레임 안에서 얼굴을 crop 하여 이미지 저장 → 원하는 얼굴 변형 적용
- 얼굴 변형시 속도 3배 이상 증가
- 다른 인물들간의 얼굴을 비교하기에 용이
개발 진행 과정
- 업로드 영상에서 face detection된 프레임 display
- 눈, 코, 입 등 정해진 부위를 성형하여 이미지에 반영
- 원하는 형태로 변경된 이미지를 영상에 적용하여 새로운 영상(이미지)으로 export
- 이미지와 영상을 display하고 조작할 수 있는 GUI 제공

  • (초반 설계) 초반에 프로그램 설계부터 (프로그램 크기, 기능 정의 등) 많은 것들을 논의한 이후에 개발에 들어갔어야 하는데 그러지 못한점. → 미팅을 초반부터 시작해서 많이 했으면 매우 좋았을듯
  • (테스트) 개발 초기단계에서 한가지 기술말고도 여러가지 기술로 테스트를 진행해보고 서로의 장단점을 파악해두고 시작하면 나중에 한곳에서 문제가 생겼을 때, 대처하기가 쉽다.
  • (멀티쓰레드) GUI 개발 시, 여러가지 기능이 동시에 작동될 수 있다는 것을 초기에 파악하고 미리 구조를 짜놨으면 더 효율적으로 진행될 수 있었을 것 같다.

- 또한, Faster R-CNN 모델(0.5FPS)보다 FPS가 월등히 빨라(45FPS) 돌발적인 상황에 가장 어울리는 모델이다.
